## 新ロードマップ構成案

### 3.1 横展開ビジネスモデル
WMG2027で開発したAIチャットボットを、コンポーネント化して横展開するビジネスモデル。

**ビジネスモデル図（SVG作成）**
- サイビット: AIチャットボットコンポーネント開発・提供
- ペタビット: 販売チャネル（既存クライアントへ展開）
- エンドクライアント: 利用・購入
- レベニューシェア: サイビット ←→ ペタビット

### 3.2 横展開先
- 関西広域連合の他イベント
- 海外のスポーツ自治体
- ペタビット社既存クライアント

### 3.3 横展開に向けた機能改善
現行チャットボットを横展開可能な汎用コンポーネントとして改善する。

  - WMG特化（○）: No.6, 7, 8, 18, 19
  - 汎用（空欄）: その他

### 新機能
- **質問候補UI表示**: AIが文脈に応じて質問候補を動的に生成し、UI上に表示して選択できる機能
  - 生成方法: 動的（AIが会話の文脈から次の質問を提案）
  - 表示位置: チャット入力欄の上または回答後
  - 候補数: 3〜5件程度

### ソート順
1. 対象（チャットボット → 管理画面 → 新規）
2. 概算費用（安い順）

## ビジネスモデル図（SVG）設計

### 構成要素
```
┌─────────────┐      ┌─────────────┐      ┌─────────────┐
│  サイビット  │      │  ペタビット  │      │   クライアント  │
│  (開発会社)  │      │  (販売会社)  │      │   (利用者)   │
└─────────────┘      └─────────────┘      └─────────────┘
```

### フロー
1. サイビット → ペタビット: AIチャットボットコンポーネント提供
2. ペタビット → クライアント: 販売・導入支援
3. クライアント → ペタビット: 利用料支払い
4. ペタビット → サイビット: レベニューシェア

### スタイル
- CLAUDE.mdのSVG作成ルールに準拠
- WMG2027テーマカラー（#ed1c2a）を使用
- 凡例を含める
- 直角線を使用
