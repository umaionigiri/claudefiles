# Naming Rules

## Directory Naming
- `docs/` subdirectories: Use `English_日本語` format (e.g., `docs/Architecture_アーキテクチャ/`)
- Code directories: Use lowercase kebab-case (e.g., `user-auth/`, `api-gateway/`)

## File Naming
- Estimate documents: `{project}_概算見積-{title}_{YYYYMMDD}.md` (Japanese filenames for client visibility)
- Config files: English lowercase (e.g., `settings.json`, `CLAUDE.md`)
- Scripts: English lowercase with extension (e.g., `generate-dashboard.mjs`)
