# DOCX 変換ルール

## Markdown 記述ルール

| 要素 | 記法 | 注意事項 |
|------|------|---------|
| 見出し | `#` 〜 `####` | Heading 1-4 に変換 |
| 表 | pipe table | `|` 区切り必須 |
| 画像 | `![](./path.png)` | alt text は空に |
| リスト | `-` or `1.` | 標準記法 |

## 改ページルール

- H2（##）セクションの直前で改ページ
- ただし H1 の直後に H2 がある場合は改ページしない

## フォントスタイル

| 要素 | フォント | サイズ |
|------|----------|--------|
| Heading 1 | Meiryo UI | 14pt |
| Heading 2-4 | Meiryo UI | 12pt |
| 本文 | Meiryo UI | 10.5pt |
| 表セル | Meiryo UI | 10pt |

## 表スタイル

- タイトル行: 背景色 `#F0F0F0`
- 罫線: 黒 (`#000000`)、単線

## トラブルシューティング

| 問題 | 原因 | 対処 |
|------|------|------|
| 日本語文字化け | エンコーディング | UTF-8 で保存 |
| 画像非表示 | パス不正 | 相対パス確認 |
| 表罫線なし | pandoc 制限 | python-docx 後処理 |
