# Claude Code スラッシュコマンド完全ガイド

$ARGUMENTS に基づいて、以下の情報から該当する部分を日本語で分かりやすく解説してください。
引数がない場合は、全体の概要とカテゴリ一覧を表示してください。

---

## カテゴリ一覧

1. **基本操作** - 会話の管理、終了、ヘルプ
2. **セッション管理** - 会話の保存、再開、エクスポート
3. **コンテキスト管理** - メモリ、コンパクト、コンテキスト確認
4. **開発支援** - コードレビュー、セキュリティ、TODO管理
5. **設定・構成** - 権限、フック、MCP、プラグイン
6. **アカウント・プラン** - ログイン、使用量、統計
7. **環境設定** - IDE連携、ターミナル、ステータスライン
8. **その他** - 診断、バグ報告、サンドボックス

## 全コマンド一覧

### 基本操作

| コマンド | 説明 |
|---------|------|
| `/help` | ヘルプを表示。利用可能なコマンド一覧を確認できる |
| `/clear` | 会話履歴をクリア。新しい会話を開始する |
| `/exit` | Claude Code を終了する |

### セッション管理

| コマンド | 説明 |
|---------|------|
| `/resume [session]` | 過去の会話を再開。IDや名前を指定、または選択画面を表示 |
| `/rename <name>` | 現在のセッションに名前を付けて識別しやすくする |
| `/export [filename]` | 会話をファイルまたはクリップボードにエクスポート |
| `/rewind` | 会話やコードを巻き戻す。変更を取り消したい時に使用 |

### コンテキスト管理

| コマンド | 説明 |
|---------|------|
| `/memory` | CLAUDE.md メモリファイルを編集。プロジェクト固有の指示を保存 |
| `/compact [instructions]` | 会話を要約して圧縮。オプションでフォーカス指示を追加可能 |
| `/context` | 現在のコンテキスト使用量をカラーグリッドで可視化 |
| `/cost` | トークン使用量の統計を表示 |

### 開発支援

| コマンド | 説明 |
|---------|------|
| `/review` | コードレビューをリクエスト |
| `/security-review` | 現在のブランチの保留中の変更に対してセキュリティレビューを実行 |
| `/todos` | 現在のTODOアイテム一覧を表示 |
| `/pr-comments` | プルリクエストのコメントを表示 |
| `/init` | CLAUDE.md ガイドでプロジェクトを初期化 |

### 設定・構成

| コマンド | 説明 |
|---------|------|
| `/config` | 設定画面（Config タブ）を開く |
| `/permissions` | 権限の表示・更新。ツールの許可設定を管理 |
| `/hooks` | ツールイベント用のフック設定を管理 |
| `/mcp` | MCP サーバー接続と OAuth 認証を管理 |
| `/plugin` | Claude Code プラグインを管理 |
| `/agents` | カスタム AI サブエージェントを管理 |
| `/model` | AI モデルを選択・変更 |
| `/output-style [style]` | 出力スタイルを設定（直接指定または選択メニュー） |
| `/add-dir` | 追加の作業ディレクトリを追加 |

### アカウント・プラン

| コマンド | 説明 |
|---------|------|
| `/login` | Anthropic アカウントを切り替え |
| `/logout` | Anthropic アカウントからサインアウト |
| `/usage` | プラン使用制限とレート制限状況を表示（サブスクリプションプランのみ） |
| `/stats` | 日次使用量、セッション履歴、連続使用日数、モデル設定を可視化 |
| `/privacy-settings` | プライバシー設定を表示・更新 |

### 環境設定

| コマンド | 説明 |
|---------|------|
| `/ide` | IDE 連携を管理、ステータスを表示 |
| `/terminal-setup` | Shift+Enter キーバインドをインストール（iTerm2/VSCode のみ） |
| `/statusline` | Claude Code のステータスライン UI を設定 |
| `/status` | 設定画面（Status タブ）を開き、バージョン・モデル・アカウント・接続状況を表示 |
| `/vim` | vim モードに入る。挿入モードとコマンドモードを切り替え |

### その他

| コマンド | 説明 |
|---------|------|
| `/doctor` | Claude Code インストールの健全性をチェック |
| `/bug` | バグを報告（会話を Anthropic に送信） |
| `/sandbox` | ファイルシステムとネットワークを分離したサンドボックス bash ツールを有効化 |
| `/bashes` | バックグラウンドタスクの一覧表示・管理 |
| `/release-notes` | リリースノートを表示 |
| `/install-github-app` | リポジトリに Claude GitHub Actions をセットアップ |

## 使用例

```
/resume                    # セッション選択画面を表示
/resume my-feature         # "my-feature" という名前のセッションを再開
/compact バグ修正に集中     # バグ修正関連の内容を保持して圧縮
/export chat.md            # chat.md に会話をエクスポート
/model                     # モデル選択画面を表示
```

## ヒント

- **Tab 補完**: コマンド入力中に Tab キーで補完できます
- **引数のヒント**: `/help` でコマンドの引数ヒントを確認できます
- **カスタムコマンド**: `~/.claude/commands/` に独自コマンドを追加できます
- **プロジェクト固有**: `.claude/commands/` にプロジェクト専用コマンドを配置できます
